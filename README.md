# Projeto de TQS

**Contexto do projeto:**
  O produto é uma aplicação online chamada “MagicLook” que serve para alugar roupas de cerimónia, permitindo que utilizadores encontrem, reservem e devolvam roupas para eventos únicos, promovendo a reutilização e reduzindo o desperdício.

**Objetivo:**
Facilitar o acesso a roupas de cerimónia para uso ocasional, garantindo processos simples, sustentáveis e eficientes tanto para clientes como fornecedores.
